# C-Fibonacci-LSB-Steganography-Tool-V2
Objection Oriented Programming based Fibonacci LSB Steganography tool with extended functionaloties

To use, first compile the code using the following command:

`` g++ main.cpp classes/*  -o project.exe ``

# Usage:
Embed:  ``./project.exe -e <input_bmp> <output_bmp> <input_txt>`` 

Extract: ``./project.exe -d <input_bmp> <output_txt>``


stb_image.h and stb_image_write.h are libraries taken form ```https://github.com/nothings/stb/blob/master/stb_image.h```
